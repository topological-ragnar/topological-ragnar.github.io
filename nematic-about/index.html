Erratic Nematics

2019

A nematic phase is a type of liquid crystal which retains orientational order while lacking translational order. They are famous for their beauty.

This three.js project grew out of a faulty implementation of a visualizer for these liquid crystals. However, I thought the unintended outcome of the initial experiment was more interesting than what I had expected to come out, so I developed it.

The image is a grid of pillars each given an internal pair of indices. Then these indices are used to define a rotation on that pillar about either the x, y, or z axes. One can choose the transformation in the menu on the upper-right.

One can navigate the scene using fly-controls:

  • w,a,s,d lateral movement
  • q,e roll
  • r,f vertical movement
  • ↑ ↓ ← → pitch and yaw

The code for this project is available on github.
